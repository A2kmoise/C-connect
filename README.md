# C-Connect

C-Connect is a modern web-based platform designed to streamline communication within an institution. It enables efficient creation, management, and distribution of announcements, ensuring that all members stay informed in real time.

## Role-Based Access Control

The system implements role-based access to maintain security and organization:

 Admin – Manages users, system settings, and announcements
 Teacher – Creates and shares announcements with students
 Student – Views and interacts with relevant announcements

## Key Features
Centralized announcement management
Real-time communication across the institution
Secure role-based access control
Responsive and user-friendly interface

## Technologies Used
Java Servlets – Backend logic and request handling
JSP (JavaServer Pages) – Dynamic web page rendering
Tailwind CSS – Modern and responsive UI design
MySQL – Database management system
